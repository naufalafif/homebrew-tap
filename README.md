# Homebrew Tap

Homebrew cask for [AgentGuard](https://github.com/naufalafif/agent-guard).

## Install

```bash
brew tap naufalafif/tap
brew install --cask agent-guard
```

## Casks

| Cask | Description |
|------|-------------|
| `agent-guard` | macOS menu bar security scanner for MCP servers and AI agent skills |
